NAME

bup-import-rsnapshot - import a rsnapshot archive

SYNOPSIS

bup import-rsnapshot [-n] [

]

SYNOPSIS

bup import-rsnapshot imports a rsnapshot archive. The timestamps for the backups are preserved and the path to the rsnapshot archive is stripped from the paths.

bup import-rsnapshot either imports the whole archive or only imports all backups for a given backuptarget.

OPTIONS

-n,--dry-rung
don't do anything just print out what would be done

EXAMPLE

$ bup import-rsnapshot /.snapshots
